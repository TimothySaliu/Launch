# Best Education Launch Game

## How to Run
1. Open Unity Hub
2. Click "Add Project" and select this folder
3. Open the project in Unity 6.3 LTS
4. Open Assets/Scenes/MainMenu
5. Press Play in the Unity Editor

## Controls
- Move: WASD or Arrow Keys
- Shoot: Space Bar

## Win Condition
Destroy 50 asteroids to win!

## Lose Condition
You have 3 lives. Lose all 3 to get a Game Over.

## Branding
Game developed for Best Education B.V.
Slogan: "We launch you into the future!"
